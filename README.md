# 🤖 Robot Avoider ESP32 Version 2 with Buzzer

## 📖 Description

Robot Avoider adalah robot berbasis ESP32 yang mampu mendeteksi dan menghindari rintangan menggunakan tiga sensor ultrasonik HC-SR04.

## 🔧 Hardware

- ESP32 DevKit V1
- HC-SR04 ×3
- TC1508A Motor Driver
- LCD I2C 16x2
- Buzzer
- 2 DC Motor
- 2 Battery 18650
